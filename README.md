# Linux-Torrents / BSD-Torrents Mirror

This repository collects **official torrents from Linux and BSD distributions**. All files are legal mirrors of open-source projects – download them via torrent, verify the checksums, and share them with the community!

## 🚀 What's here?
- **Linux Distros**: Ubuntu, Fedora, Debian, Arch, Mint, Pop!_OS and more.
- **BSD Variants**: FreeBSD, GhostBSD, OpenBSD, DragonFlyBSD, pfSense.
- **ISO Torrents**: Current releases with official checksums and installation guides.
- **Goal**: Fast, decentralized downloads through P2P sharing – no central servers!

**Current Highlights:**
- [GhostBSD-25.02-R14.3p2.iso.torrent](GhostBSD-25.02-R14.3p2.iso.torrent) – FreeBSD 14.3 with MATE (~3.6 GB)
