# Linux-Torrents / BSD-Torrents Mirror

Dieses Repository sammelt **offizielle Torrents von Linux- und BSD-Distributionen**. Alle Dateien sind legale Mirrors von Open-Source-Projekten – lade sie via Torrent herunter, überprüfe die Integrität und **seed langfristig**, um die Community zu unterstützen!

## 🚀 Was gibt's hier?
- **Linux-Distros**: Ubuntu, Fedora, Debian, Arch, Mint, Pop!_OS und mehr.
- **BSD-Varianten**: FreeBSD, GhostBSD, OpenBSD, DragonFlyBSD, pfSense.
- **ISO-Torrents**: Aktuelle Releases mit offiziellen Checksums und Install-Anleitungen.
- **Ziel**: Schnelle, dezentrale Downloads durch P2P-Sharing – keine zentralen Server!
